-API Laravel
Para instalar e rodar a api execute os seguintes comandos:

    cd braviapi
    
    composer install --ignore-platform-reqs
  
    ./vendor/bin/sail up -d

-Front Angular

Para instalar e rodar o front execute os seguintes comandos:

    cd bravifront
    npm install
    ng serve
